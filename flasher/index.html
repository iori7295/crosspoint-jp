CrossPoint JP Flasher

ブラウザからXteinkデバイスにファームウェアを書き込めます

Web Serial APIに対応していません。
Chrome 89+ または Edge 89+ をご利用ください。

ファームウェアの選択

最新の変更を含む開発ビルド。
テスト済みの安定リリース。
対応ブラウザ: Chrome/Edge HTTPS接続が必要です

使い方

  1. XteinkデバイスをUSB-Cで接続
  2. ファームウェアを選択し「インストール」をクリック
  3. シリアルポートを選択
  4. 書き込み完了まで待機（約2分）

注意事項

  • 対応: Chrome 89+, Edge 89+（Safari/Firefox非対応）
  • 初回書き込み時は既存データが消去されます
  • 書き込み中はケーブルを抜かないでください
